# 移除播放按钮边框效果 - 实现计划

## [x] 任务 1: 移除播放按钮的边框样式
- **优先级**: P0
- **依赖**: 无
- **描述**:
  - 修改 `play-btn` 类，移除 `border: 6rpx solid rgba(255, 255, 255, 0.3);` 样式
  - 确保按钮仍然保持美观的视觉效果

  - 按钮的其他样式保持不变
- **测试要求**:
